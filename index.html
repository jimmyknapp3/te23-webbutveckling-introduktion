Html Text element

För att formatera text med HTML så kan jag använda olika rubrik-element, heading på engelska. För text stycken så används paragrafer. Här nedan följer ett antal exempel på HTML text-element. Du kan läsa mer om HTML text-element hos Mozilla
  • <h1> - heading1 elementet, största rubriken
  • h2 - heading 2 elemement, andra rubriknivån
  • <p> - paragraph, ett textblock med radbrytning efteråt
  • <ul> - ul står för unordered list
  • <a> - a används för att länka
